What it does

AI roasts

Ollama (default) or Gemini, with 60+ template fallbacks injected with internet-brainrot vocabulary. Goes hard.

Score & ranks

Bronze → Diamond. Every commit, push, and merge conflict shifts your score in real time.

20 achievements

Unlocked from real Git behaviour. Mostly humiliating. Occasionally aspirational.

Source control, but worse

In-sidebar SCM with AI-generated commit messages. Pick clean (Conventional Commits) or savage (toxic coach).

Weekly hygiene report

7-day metrics — force pushes, pushes to main, conflicts, streaks — each with a roast caption.

Shareable rank card

Export your rap sheet as a 1200×630 SVG. One click to tweet it.

Install

Three ways. Pick your fighter.

One-click

Hit the install button up top, or open this link in VS Code:

vscode:extension/git-gud.git-gud-vscode

From inside VS Code

Open the Extensions panel and search:

git gud coach

Command line

If you live in the terminal:

code --install-extension git-gud.git-gud-vscode

Building from source? See the repo.
